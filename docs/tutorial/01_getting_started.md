# Getting started

- install ndx-framework, only needs doing once  
```bash
npm install -g ndx-framework
ndx-framework --init
```
- make a new app  
```bash
ndx-framework --create --appname=tutorial
```
- navigate to http://localhost:3000 to see your app running
